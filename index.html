Déploiement WordPress Haute Disponibilité

Docker Swarm · GlusterFS · Prometheus · CI/CD · Dokploy

📦 Objectif du projet

Ce projet consiste à déployer une infrastructure WordPress complète, scalable et hautement disponible à l'aide de :

Docker Swarm GlusterFS Prometheus Alertmanager CI/CD Dokploy

🔧 Composants

  • Cluster de 3 VM : VM1 (Manager), VM2, VM3
  • Stockage partagé via GlusterFS (volumes WordPress & MariaDB)
  • Monitoring avec Prometheus & Alertmanager
  • Déploiement continu via Dokploy connecté à GitHub
  • Infrastructure sécurisée et redondante (compute + storage)

🌐 Accès rapides

  • WordPress : wordpress.vm1.courses.studalya.net
  • Dokploy : dokploy.vm1.courses.studalya.net
  • Prometheus : vm1:9090
  • Alertmanager : vm1:9093
  • Portainer : vm1:9000

🧪 Technologies utilisées

  • Docker Engine + Swarm Mode
  • GlusterFS pour la persistance des données
  • CI/CD avec GitHub Actions ou webhooks Dokploy
  • Alertes envoyées via Alertmanager vers Discord/Mail
  • Documentation centralisée via Bookstack
